# SIDEQUEST PICKER 🎰

Adventure Quest Picker mit Slot-Machine Animation.

## Dateien

```
index.html      → Einstiegspunkt
styles.css      → Design (Lime-Grün Dark Theme)
app.js          → App-Logik + alle 22 Quests (DE/EN)
roulette.js     → Slot-Machine-Bibliothek (jQuery Plugin)
```

## Lokal öffnen

Direkt `index.html` im Browser öffnen — kein Build-Schritt nötig.

## GitHub Pages

1. Alle 4 Dateien in ein Repo pushen
2. Settings → Pages → Branch: `main` / Root
3. Live unter `https://USERNAME.github.io/REPO/`

## Features

- 🎰 Slot-Machine mit echter Scroll-Animation (roulette.js)
- 🌍 DE / EN Sprachschalter (oben rechts im Header)
- 🔧 Filter: Kategorie, Dauer, Ort, Gruppe, Level
- ✅ Done · Skip · Speichern mit Verlauf
- ➕ Eigene Quests erstellen (mit DE + EN Titel)
- 💾 Alles im localStorage gespeichert
